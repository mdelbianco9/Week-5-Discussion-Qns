What is Bootstrap?

Bootstrap is a giant collection of handy, reusable bits of code written in HTML, CSS, and JavaScript. It’s also a front-end development framework that enables developers & designers to quickly build fully responsive websites.

Why do we like Bootstrap?

Bootstrap saves you from writing lots of CSS code, giving you more time to spend on designing webpages. It makes our lives as developers easier.

When using Bootstraps grid system (columns), what containing element class or classes do we need?

A div with the class of "container". And a div with the class of "row".

Whats the difference between Bootstraps container and container-fluid?

“container” has horizontal margin and doesn’t take up the full width of the viewport. “container-fluid” takes up the full width of the viewport.

“container” uses break points to resize and “container-fluid” constantly readjusts as the viewport is resized.
